# 装甲板检测器

## 项目简介
这是一个基于OpenCV的装甲板检测系统，能够从视频流中检测红色装甲板，并使用PnP算法计算装甲板的三维位置和姿态信息。

## 主要功能
1. 图像预处理（高斯模糊、形态学操作）
2. 红色区域检测（HSV颜色空间）
3. 灯条检测与筛选
4. 装甲板匹配
5. PnP解算（获取旋转向量、平移向量、欧拉角、四元数）
6. 实时视频处理

## 系统架构
- `LightBar`类：灯条检测与表示
- `Armor`类：装甲板检测与表示
- `PnPSolver`类：PnP解算器
- `ArmorDetector`类：主检测器
