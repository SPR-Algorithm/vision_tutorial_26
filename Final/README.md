## 大作业

使用所学的知识完成，装甲板detector的设计与实现。
### 任务描述
设计并实现一个装甲板检测器，能够在给定的图像或视频中检测并标记出装甲板的位置。装甲板通常具有特定的颜色
和形状特征，检测器需要能够识别这些特征并准确定位装甲板。


### 要求
1. **图像预处理**：实现图像的预处理步骤，如去噪、增强对比度等，以提高检测的准确性。(opencv图像处理)
2. **特征提取**：设计并实现特征提取算法，能够识别装甲板的颜色和形状特征。
3. **检测算法**：实现装甲板的检测算法，能够在图像中定位装甲板的位置。
4. **结果展示**：在图像或视频中标记出检测到的装甲板，并展示检测结果。输出目标的t_vector,r_vector，旋转矩阵，欧拉角，四元数信息。(pnp结算)
5. **类的封装**：将检测器的各个功能模块封装成类，便于管理和复用。如armorDetector类，armor结构体，灯条结构体等。（oop编程思想）
6. **代码规范**：代码需符合良好的编程规范，包含适当。
7. 提交到自己的learn_log/目录下。

加分项：
- 使用神经网络来检测装甲板.(可以自学了解模型的使用，范例见lecture4中的onnx文件)
